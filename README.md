# gotools

`gotools.nvim` provides some code action functions that can be integrated with null-ls or can be used separately

## Install
* Requires Neovim >= 0.7
* gotests
* gomodifytags
Recommend `[mason.nvim](https://github.com/williamboman/mason.nvim)` to manage binary tools
